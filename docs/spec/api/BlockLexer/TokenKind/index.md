# API: TokenKind 列挙体/クラス

## 概要

トークンの型を表す。
デフォルトの型 ( `int` ) を使用する。

## バリアント

|       名前       |     内容     |
| :--------------: | :----------: |
|        ID        |      ID      |
|      Number      |     数値     |
|      Space       |   スペース   |
|      String      |    文字列    |
| StringInBrackets | 括弧内文字列 |
|      Symbol      |     記号     |
